# vimrc
A shell script to deploy my vim configuration

## preview
![preview](preview.jpg)

## install
The installation will overwrite your `.vimrc` file, be sure to backup your existing vim configuration.

```
$ git clone https://github.com/modood/vimrc.git
$ cd vimrc
$ ./deploy.sh
```
